# hoppyRabbit
A simple game created using Pygame, where the player controls a rabbit character to collect edible food while avoiding toxic food for the rabbit.

## Installation

```bash
$ git clone https://github.com/hatatwit/hobbyRabbit.git
$ cd hobbyRabbit
$ pip install pygame
$ python main.py

```
